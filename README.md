# Trip Advisor Clone

This is a simple Trip Advisor homepage clone created using HTML and CSS.

The main purpose of this project is to practice:
- HTML page structure
- CSS styling
- Flexbox layout
- Hover effects
- Simple animations
- Responsive design

## Features
- Header with logo
- Search bar
- Navigation menu
- Hero section with animated title
- Filter buttons
- Popular places cards
- Footer with social media icons

## Technologies Used
- HTML
- CSS
- Flexbox
- Font Awesome
